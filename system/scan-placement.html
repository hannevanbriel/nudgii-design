Monetisatie
Scan-logica
Free tier
Eerste onboarding-scan gratis
Eerste scan tijdens onboarding is altijd gratis — geen paywall-hint anywhere.

2 scans/maand post-onboarding. Paywall verschijnt bij maandlimiet (est. week 3).
Pro tier
Onbeperkte scans
Copy bij paywall trigger: "Je hebt je 2 scans gebruikt — upgrade voor onbeperkt."

Nooit getoond tijdens onboarding. Paywall pas bij eerste keer dat maandlimiet geraakt wordt in dagelijks gebruik.
Conversie-benchmark
Scan = acquisitiekosten
Gebruikers die scannen tijdens onboarding converteren op ~2× rate (Planta benchmark).

Eerste scan = CAC, geen lopende kosten. Altijd gratis houden tijdens onboarding.
API-kosten
Plant.id + Claude vision
Duurste API-call in de app. Free tier 2/maand is haalbaar op schaal.

Plant.id voor planten, Claude API (claude-sonnet-4-6) voor alles overig — single JSON call met item_type, brand, model, tasks[].
Regels
Vergrendelde scan-beslissingen
Eerste onboarding-scan: altijd gratis
Geen paywall-hint anywhere tijdens onboarding. Eerste scan = acquisitiekosten, geen running cost.
Paywall trigger: maandlimiet in dagelijks gebruik
Paywall verschijnt de eerste keer dat een gebruiker hun maandlimiet raakt in dagelijks gebruik — nooit eerder.
Copy-toon: helpful, not commercial
Consistent met nudgii voice. "Je hebt je 2 scans gebruikt — upgrade voor onbeperkt." Nooit opdringerig, nooit eerder dan noodzakelijk.
S-02b blokkeert v1.0 niet
Het volledig scan-scherm binnenin Path A is v1.1. v1.0 launch gaat door met S-05 camera icon + FAB scan accent.
