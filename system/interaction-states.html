Loading states
Spinners · skeletons · when to use each
Spinner · default · 20px
Spinner · small · 14px
Spinner on CTA button
Skeleton cards · shimmer · initial data load
Spinner: use for operations <3 seconds (saving, AI processing, auth). Spin duration 700ms linear.
Skeleton: use for initial screen data load only. Shows 2–3 placeholder cards. Shimmer direction: left to right, 1400ms infinite.
Never stack spinners. Only one loading indicator visible at a time. If multiple things are loading, use the skeleton for the primary content area.
Flutter: CircularProgressIndicator.adaptive() with custom color and strokeWidth: 2.5. For skeletons, use shimmer package or a custom AnimationController with a gradient sweep. Never use LinearProgressIndicator — it reads as a determinate loading bar, which is wrong for these interactions.
