Rules for new items

When engineering adds a new curated item type

  • 1. Assign to one of the 6 categories (Home, Vehicle, Garden, Subscriptions, Health, Pets)
  • 2. Define 2-5 default tasks with cadence, DIY/Pro/Appointment type, typical cost
  • 3. Mark consumable tasks with ⬡ (affiliate link opportunity)
  • 4. Suggest a personality name (or "(user's name)" for pets, "(none)" for subscriptions)
  • 5. Write "Why it matters" and "How to do it" content per task
  • 6. Define detail fields (brand, model, area, provider, etc.)

AI-generated templates (unlisted items)

When a user adds an item not in this list, the Claude API generates a template on the fly. Full spec: ai-template-generation.html. Popular AI templates (20+ uses, >80% acceptance) get promoted to this curated list after manual review.

Personality name rules

  • Short: one word preferred, two max
  • Obvious: should make you smile, not think
  • Never the default: generic item type is always the default name
  • Pets: suggest "(user's pet name)" instead of a nudgii nickname
  • Subscriptions/Health: no nickname (these are serious, not playful)
  • Offered once as a suggestion chip, always editable
