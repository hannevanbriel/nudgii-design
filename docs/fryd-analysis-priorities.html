Area nudgii Fryd
Sign-up timing ✓ After celebration (S-05 State 3). User has seen value. ✗ Before any value is shown. High drop-off.
Onboarding tutorial ✓ No tutorial needed. UI is self-explanatory. ✗ 7-screen tutorial that users skip immediately.
GDPR consent ✓ Unchecked by default. Explicit opt-in. ✗ Pre-ticked marketing consent. Legal risk.
Celebration moment ✓ User-controlled tap, plum overlay, spring animation. ~ 1.2s auto-advance. Too fast to land.
Monetization model ✓ Freemium, no trial, no tricks. 10 items free forever. ~ 7-day trial. Too short for maintenance apps.
Urgency language ✓ "Waiting for you," time-since-done. No guilt. ✗ "Overdue" badges. Creates anxiety.
Analytics privacy ✓ Mixpanel first-party only. No ATT until ads. ~ ATT prompt on first launch. Unnecessary friction.
