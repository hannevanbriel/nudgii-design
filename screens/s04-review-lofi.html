Zones
Review screen
A+BStatus bar + progress bar
Standard iOS notch + status bar. Progress bar: full-width, edge-to-edge, 3px, colorCta #9B7FD4 at 60% (S-04 = step 3 of 5). Outside SafeArea. Grows by 8% per selected item. Animates via AnimatedContainer.
CHeader
DM Serif Display 20px. "Here's what nudgii found." Action-oriented: nudgii tells the user what it discovered. Subtitle: DM Sans 11px italic colorMidAccessible. "Tap a card to review its tasks. Adjust anything later." One serif moment per screen.
DCategory chips
Same chips as S-03. Tinted bg + category color text. "All" active by default with colorCta outline. Single scrollable row (nowrap + overflow-x:auto).
Home: rgba(122,90,80,0.13) bg, #7A5A50 text.
Vehicle: rgba(74,106,130,0.13) bg, #4A6A82 text.
Garden: rgba(74,122,64,0.13) bg, #4A7A40 text.
Subscriptions: rgba(106,80,133,0.13) bg, #6A5085 text.
Active category: adds 1px solid categoryColor outline. Filters tile grid to show only that category.
ETile grid, 2-column layout
2-column grid per category section. Category header: 9px, 600 weight, uppercase, colorMidAccessible + 6px colored dot + item count.
Category-tinted backgrounds at 12% opacity. Bare colored SVG icons (no container). Tile shows: icon, item name, task count, chevron.
Chevron tap: expands inline drawer below the tile. Only one drawer open at a time. Closing animation: 250ms ease-in-out.
EInline drawer
White panel expands below the tapped tile, spanning full grid width. Shows task list for that item.
Each task row: colored first-task dot (category color), task name, frequency label (right-aligned, colorMid), delete button (x icon).
Pro badge: rgba(155,127,212,0.12) bg, colorCta text. Shown on tasks locked behind Pro tier. Delete button hidden on Pro-locked tasks.
FAdd item row
Dashed border (1.5px, rgba(26,22,18,0.16)), 12px radius. Plus icon in subtle circle + "Add another item" text.
At free tier limit (10 items): text changes to "Free plan limit reached". Plus icon becomes lock icon. "Go Pro" link replaces tap action.
GFree tier banner
Always visible in sticky bottom zone. 10-pip indicator showing item count. White surface, 1px border, 10px radius.
Pip colors: items 1-7 = colorCta purple. Items 8-9 = colorApricot warning. Item 10 = all pips filled. "Go Pro" link always visible, right-aligned.
Text: "X of 10 free items". Updates live as items are added or removed.
HCTA button
Full width, 44px, radius 100px. "Looks good →". colorCta bg, colorCtaFg text. Always enabled (user can proceed at any point).
Pressed: scale(0.97), 150ms ease-out. Shadow: 0 4px 16px rgba(155,127,212,0.30).
-Navigation
iOS swipe gesture only. No visible back button. Swipe right to return to S-02 (Path A) or S-03 (Path B).
