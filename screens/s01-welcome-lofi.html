AStatus bar
Standard iOS status bar. Time left, signal right. Color: var(--ink) on var(--cream) background. Not customisable — OS element.
Bii mark + free badge
ii mark — left aligned, 20px height.
Left pill: var(--ink) — right pill: cta 45% opacity — dot: apricot-dk.
Mirrors the home screen icon exactly: same three colours.
Free badge — right aligned.
Background: #EAF3DE (sage-lt) · radius: 100px
Dot: #3B6D11 (sage) · text: "Free · no credit card"
Font: 9px weight 500, color #3B6D11
CHeadline + subtitle
Headline — DM Serif Display · 33px · lh 0.97 · ls -0.55px · var(--ink)
Copy: "Your stuff won't maintain itself."
Always this text — not locale-dependent.
Subtitle — DM Sans 300 italic · 12px · var(--mid) · lh 1.55
Copy: "But we can remind you."
DPreview items — item row style
3 item rows matching the dashboard visual language. White card on cream, border. Left: category icon chip (24×24px, category-colored). Item name 10px/500 + status label 8.5px. Overdue = terra. Never mini-tiles, never columns — items always look like items throughout the app.
EDivider
Two horizontal lines (1px · border-md) with centered label text.
Copy: "How do you want to start?" · 10px · var(--mid)
Separates the preview section from the action choice.
