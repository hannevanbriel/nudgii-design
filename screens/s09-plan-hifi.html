Screen structure
Layout Plan tab shell
Fixed header zone (title + export icon + chip row), then a scrollable content area with monthly sections. Title: Noto Serif 24px, letter-spacing: -0.02em. Export icon top-right: 32x32px, icon-only, mid color, no bg. Chip row: horizontal scroll, nowrap, 24px horizontal padding, 6px gap. No second line of chips ever.
Export (Pro only)
Decision Icon-only, toned down
Export is a Pro-only feature. Icon-only button (Phosphor DownloadSimple, 16px, mid color). No label, no fill, no border. 32x32 tap target inside 44x44 accessibility zone. Tapping opens export bottom sheet (PDF / Calendar / Share). Free users see Pro upgrade prompt instead.

Previous plum pill competed visually with the FAB. Two plum elements dilute the FAB's primacy. Export is secondary, FAB is primary.

Flutter: IconButton(icon: PhosphorIcons.downloadSimple, color: colorMid, onPressed: showExportSheet). Gate behind Pro subscription check.
Filter chips
Chips Category-colored active states
"All" active: background: var(--plum), color: #FAF8FF. Category pills active: their own category color fill + cream text. Home = #C09088, Garden = #8C9B80, Vehicle = #8E9A94, Subs = #A69BBF. Inactive: outline-var border, ink text, transparent bg. Tap toggles filter, one active at a time. "All" is always first and default. No second line of chips, ever.

Flutter: Custom chip widget. "All" active: backgroundColor: colorCta. Category active: backgroundColor: categoryColor. All active chips: foregroundColor: #FAF8FF, no border. Inactive: transparent bg, BorderSide(color: outlineVar), ink text.
Month header
Month header Serif name + task ratio circle
Layout: [circle] [month name]. Circle: 36x36px, LEFT of name. Month name: Noto Serif 18px, ink, flex: 1. Circle: SVG with track (surface-high, 2.5px) + progress arc (var(--plum), 2.5px, round cap, rotated -90deg). Center label: 9px 600 plum, ratio format ("2/5"). When 0 done: track only, hint color label.
Spec Task ratio, not percentage
Circle shows "2/5" not "40%". Matches My Stuff progress ring pattern. Recurring tasks count as 1 occurrence per month. SVG arc: stroke-dasharray="[87.96 * (done/total)] 87.96".

Flutter: CustomPaint with two drawArc calls. Stack with centered Text('$done/$total').
Task cards (expand/collapse)
Decision Same interaction as Today
Plan task cards use the identical expand/collapse pattern from Today. If it looks the same, it must behave the same. Users learn one pattern, not two.

Collapsed: date column + icon circle + title/subtitle + optional badge + chevron (12px, 0.3 opacity, points down).
Expanded: chevron rotates 180deg. Reveals "Why it matters" block + action row (Skip / Snooze / Done). Accordion: expanding one card collapses any other open card in that month.

Date column: 40px, day abbrev (9px 500 hint uppercase) + date number (16px 600 mid). Icon circle: 32px, category wash bg. Body: 13px 500 title, 10px mid subtitle. Badge (optional): apricot "waiting Xd" for past-window tasks.
Expanded "Why it matters" + actions
"Why it matters" block: surface-low bg, 8px radius, 12px 14px padding. Label: 9px 600 uppercase, category color. Body: 11px mid, 1.5 line-height. Tappable: opens full task detail as bottom sheet (see below).

Action row: right-aligned. Skip (11px hint, no bg) + Snooze (11px ink, surface-low pill) + Done (11px, category-color pill with check icon). Done uses the task's category color, not plum. Plum = app-level actions only.
Task detail: bottom sheet
Decision Sheet replaces "Details >" link
No "Details >" text link. Instead, tapping the "Why it matters" block or the icon + title area in expanded state opens S-13 task detail as a modal bottom sheet that slides up (300ms ease). The list stays visible behind a dimmed scrim.

Why this works: The sheet is spatial. You always see the Plan list behind it. Swipe down to dismiss. You never lose context about where you are. No page navigation, no back button ambiguity. This is the Apple Maps / Stocks sheet pattern.

Sheet content: drag handle (4px, 36px wide, surface-high, 6px top margin), then S-13 content: task identity, full "Why it matters", instructions (expandable), completion history, CTA bar (Snooze + Done).

Flutter: showModalBottomSheet(isScrollControlled: true, builder: (ctx) => DraggableScrollableSheet(...)). Shared element: category icon circle animates from card to sheet header.
Swipe gestures
Decision Swipe right-to-left for actions
Swiping a task card from right to left reveals action buttons behind the card. Order from RIGHT to LEFT (trailing edge first):

1. Done (category color fill, white check icon, widest)
2. Snooze (surface-high bg, ink text)
3. Skip (subtle gray bg, mid text)

Full right-to-left swipe = mark Done (the most common action). This matches iOS Mail / Reminders swipe pattern. Applies to both Today and Plan tabs.

Flutter: Dismissible with confirmDismiss, or use flutter_slidable package for the 3-action reveal. ActionPane(motion: DrawerMotion(), children: [done, snooze, skip]).
Done tasks on Plan
Decision Stay inline, visually muted
Done tasks stay in their calendar position, muted to 45% opacity with a strikethrough title. A 20px category-colored check circle replaces the chevron. No expand on done tasks. The month progress circle updates (2/5 becomes 3/5).

Why not remove or move? Plan is a timeline. Temporal position matters. Removing "Check tyre pressure" from April 14 creates a gap that confuses scanning. The user expects to see all tasks for a month, including completed ones.

Undo: 5s toast with undo action (matching Today behavior). Tapping a done task un-does it (restores active state).

Flutter: AnimatedOpacity(opacity: task.isDone ? 0.45 : 1.0, duration: 350ms). Title: TextDecoration.lineThrough. Check circle: 20px Container(decoration: BoxDecoration(shape: circle, color: categoryColor)) with centered check icon.
Row spacing Gap, not borders
Rows sit in a Column with gap: 2px. No dividers, no border-bottom. Surface color between rows is the separator. Pressed state: rgba(49,51,44,0.03). Never wrap multiple rows in a single card container.
Seasonal quote card
Quote card Style
Appears between month sections (after the last row of a month, before the next month divider). Background: surface-low. Border-radius: 12px. Padding: 14px 16px. Quote text: Noto Serif italic, 13px, ink, line-height: 1.55. Attribution line below: 9px 500 apricot, uppercase, letter-spacing: 0.04em. Not dismissible. Content is seasonal and month-specific, stored server-side, rotated per locale.
Month divider
Divider Between months
1px horizontal line using var(--surface-high) color. 24px horizontal margin (matches screen padding). Appears above each month header (not above the first month). Creates clear visual grouping between months without heavy borders.
States
State 1 Default (has tasks)
When: user has items with tasks across upcoming months.

Full timeline rendered: month headers, task rows, seasonal quote cards, editorial closer. Filter chips active. Filter icon available. Completion circles show real progress per month.
State 2 Empty (no tasks)
When: user has no items, or items with no tasks scheduled.

Screen title still visible. Filter icon grayed out, non-interactive. Chips hidden. Centered empty state: calendar icon (64px circle, surface-low bg) + Noto Serif headline "Nothing planned yet." + 13px mid sub + plum pill CTA "Add an item". CTA opens FAB action sheet.
Editorial closer
Editorial Bottom personality line
Centered. Noto Serif italic, 12px, hint color, 0.55 opacity. Appears after the last month section. Rotates from a set of brand voice one-liners: "A house maintained is a house loved." / "Your future self will thank you." / "Still here. Always watching your gutters." Never shown in empty state.
Nav bar
Nav bar Plan tab active
Solid surface background, no glassmorphism. Slight upward shadow: 0 -2px 12px rgba(49,51,44,0.04). Active tab (Plan): var(--primary) on both SVG stroke and label text. No background shape behind the active tab. Inactive tabs: var(--hint) color. FAB: right-aligned, 46x46px, 14px radius, plum bg.
Flutter implementation
Flutter Standard block
Mockup base: 390x844 (iPhone 15).
SafeArea: yes, wraps content area.
Scroll: ListView for Plan content zone only.
Bottom zone: nav bar 84px, fixed, outside scroll.
FAB: Positioned, right: 24, bottom: 18.
Task cards: AnimatedContainer for expand/collapse (300ms ease). Accordion: expandedTaskId state. flutter_slidable for swipe actions.
Detail sheet: showModalBottomSheet(isScrollControlled: true) with DraggableScrollableSheet.
Done state: AnimatedOpacity(0.45, 350ms) + TextDecoration.lineThrough.
Month grouping: SliverList with header widgets. CustomPainter for ratio circle.

Snooze: Card mutes (title/icon at 0.5 opacity), date column updates to snoozed date, "Snoozed to [time]" label in plum below meta. Mini confetti burst (8 particles, category colors, 600ms spring). Toast with witty escalating copy + Undo. Undo restores original date + removes snooze label.

Month ratio: excludes hidden cards. Skipped (display:none) cards excluded from total count. Ratio shows done/visible, not done/all.

Skip: AnimatedContainer height 0 + opacity 0. Task not deleted, task.last_skipped_at set, task.next_due_at recalculated. Undo: restore height/opacity, clear skip fields.
Snooze: task.snoozedUntil field. Card stays in list, muted. Date column shows snoozed date. Confetti: 8 AnimatedPositioned particles with spring curve.

v1.0 · 2026-04-02 · ADDITIVE
Initial hi-fi for Plan tab. New standalone file: screens/s09-plan-hifi.html. Extracted pattern from Precision Warmth design system into full hi-fi with annotation sidebar.

Content: Three month sections (April, May, June) with serif month names, SVG completion circles, task rows using date column + 32px category icon circle + title + item. Seasonal quote cards between months. Editorial closer at bottom.

Filter chips: All (default active), Home, Garden, Vehicle, Subscriptions. Horizontal scroll row. Active state: plum border + plum text + plum-wash bg. Single active chip at a time.

No expand on task rows: Plan tab task rows navigate to task detail (S-13). Chevron-right signals navigation, not expansion. This is a deliberate departure from Today tab where rows expand in place.

Two states: Default (has tasks across months) and Empty (no items or tasks). Empty state shows centered calendar icon + headline + sub + plum pill CTA.

Dark mode: Supported via [data-theme="dark"] on phone element. All tokens update. Category washes lift to 15% opacity.
Flutter notes:
Use ListView for scrollable content zone, not SingleChildScrollView. Group tasks by month: Map<DateTime, List<Task>> keyed by DateTime(year, month). Completion circle: CustomPainter with two drawArc calls (track + progress), not CircularProgressIndicator. Task row: InkWell + Row + Navigator.push(TaskDetailRoute). No ExpansionTile. Filter chips: SingleChildScrollView(Axis.horizontal) + custom chip widgets with selectedCategory state. Category icon circle: 32px Container with BoxDecoration(shape: BoxShape.circle).
