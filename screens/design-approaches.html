How much color is too much?

Three approaches to the category color problem. How Apple and Anthropic would each solve it. Plus: celebration screens for Options A and C, and apricot as the dark mode accent color.

The problem in one sentence: nudgii's dashboard can show 10 differently-colored cards simultaneously. That's not a color system — it's a festival. The CTA purple loses dominance because it's competing with 9 other hues for the user's attention.

What Apple would do: Ruthless restraint. The neutral surface (cream/white) dominates at 80%+ of the screen. Color lives inside icon containers, never on card backgrounds. Every card is the same material; color is a label, not a decoration. iOS Health, Reminders, and Calendar all do this. You can have 50 categories and the screen never feels loud.

What Anthropic would do: Personality through copy and proportion, not color saturation. A very faint hue whisper on the card surface — 12-15% opacity — so the category reads but the card doesn't shout. The cream background still dominates. Color is atmospheric, not structural.

The rule both agree on: If every element is colorful, nothing is colorful. Reserve full saturation for the one thing you want the user to act on. For nudgii, that's the CTA purple.

160-30-10: cream/dark bg = 60%, category color = 30%, CTA purple = 10%. Bold fills push category color to 80%. The 10% rule breaks.
2Daily use penalty: You see the dashboard every day. A screen that's exciting on day 1 becomes exhausting by day 30. Calm scales; bold doesn't.
3Hierarchy first: The user's job is to read task names and act. Color should support scanning, not replace hierarchy. If you removed all color, does the task name still read clearly? It should.

Dark mode · accent color

Purple vs apricot for "Martijn."

The action color (CTA, FAB, filter pills, progress bar) stays purple in dark mode — it's the functional system color. The question is only for the greeting name: a decorative, serif, italic moment. Apricot reads warmer and more personal against the warm dark background.
