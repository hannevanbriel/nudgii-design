S-06 · Celebration state (plum)
Shell + layout
Rule Two-screen sequence
S-06 has two states: AHA (cream, 80% progress) shows the generated schedule. Celebration (plum, 85% progress) protects the emotional peak after the user taps "Open my schedule". Sequence: S-05 review, S-06 AHA, S-06 celebration, S-07 push, S-08 SSO, S-09 dashboard.
Decision AHA before auth
The schedule is shown before the user creates an account. This anchors value before asking for commitment. The user sees exactly what nudgii built for them, then saves it. "Save my schedule" is the CTA, not "Create account."
S-06 AHA content
No overdue colors
At onboarding, nothing is overdue. All dates use colorApricotDk (soonest task) or colorMidAccessible (further out). Never terra/overdue at this point. Overdue appears only after a due date passes on the dashboard.
Value box
CTA purple tinted background, rgba(155,127,212,0.06). Reinforces the core promise: "You told us what. nudgii figured out when, and why it matters." One sentence, no bullet points, no icons.
S-06 celebration state
Decision Personalized humor
Headline adapts to the user's primary item (first or most relevant). "Your boiler is relieved." / "Your dishwasher just exhaled." / "Your car is quietly grateful." Flutter: string interpolation from user's item list, with a lookup table of witty lines per item type.
First nudge pill
Glass-morphic pill: rgba(255,255,255,0.06) bg, 0.5px white border. Shows "First reminder" label + apricot dot + concrete date. Anchors the value with a specific, near-future date the user can visualize.
Confetti dots
10 CSS-positioned dots using category colors at low opacity (0.14 to 0.50). No gradients, no images. Sizes 3-6px. Flutter: Positioned widgets in a Stack. Dots are static, not animated.
Rule Never auto-advance
S-06 always requires a user tap to proceed. "Open my schedule" CTA, never auto-navigate. "Tap when you're ready. No rush." reassures without pressuring. Flutter: no Future.delayed navigation.
Rule Spring animation
On celebration state load: ii mark scales from 0.6 to 1.0, headline translates from +20px to 0. Duration: 400-450ms, cubic-bezier(0.34,1.56,0.64,1). This is the only theatrical motion in the app. Confetti dots are static.
