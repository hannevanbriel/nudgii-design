S-06 · auth design
★
"Save your schedule" — not "Create account." Reframes auth as protecting something the user already built. Higher perceived value, lower resistance.
↓
Apple first, Google second — per spec. Email toggle reveals inline below Google button, no separate screen. Magic link only — no password field.
★
Three-part reassurance line at the bottom: "Free forever for 10 items · Cancel anytime · No card needed." Addresses every objection in one line.
S-06 · email state
N
Email toggle is a text link — "or use email instead." Tap reveals the email input + CTA inline. No navigation, no separate screen, no modal.
✓
Magic link only. One tap in email — no password. Copy: "One tap in your email — no password needed." Reduces friction to near zero.
★
Email input focus state: colorCta border + subtle shadow ring. Matches the CTA colour system — the active element always speaks in purple.
S-07 · completion rules
↓
Never auto-advance. CTA "Open my schedule →" is required. User must tap — per decisions log. Gives them a moment to feel done.
★
Summary pill ("2 items · 7 tasks ready · first nudge Oct 20") creates a sense of accomplishment without being verbose. One line, scannable.
★
Witty sub: "Your dishwasher is already taking notes." Earned humour — the onboarding is complete, so a light joke lands well. Not on an error, not on a payment screen.
↓
Cream background — not plum. Plum is reserved for the task done overlay only. This is a warm, grounded moment — same cream as the rest of onboarding.
S-09 · dashboard shell
N
Floating pill tab bar — custom widget, NOT Flutter BottomNavigationBar. 3 tabs: Tasks (active) · Ask · Profile. Active = full ink + 3px colorCta pip.
N
FAB — 46×46px, borderRadius 15px (rounded square, not circle), colorCta. Adjacent to pill bar. Plus icon, rotates 45° to × when action sheet is open.
★
Greeting with name in apricot — DM Serif Display 25–27px, name rendered in colorApricot. One serif moment, one screen.
★
Section labels — Overdue / This week / Coming up. 9px uppercase, colorMid, letter-spacing 1.8px. The only uppercase text in the app.
↓
No left border accent. Status lives in date label colour only — terra = overdue, apricot-dk = due soon, mid = ok. Cards are identical in shape, different in date colour.
