Structure + shell
Rule Progress bar
60% fill (step 3 of 5). Full-width, edge-to-edge, outside SafeArea. 3px height, colorCta fill. Track: rgba(26,22,18,0.08). Flutter: LinearProgressIndicator inside ClipRRect(borderRadius:100px).
Rule Shell pattern
SafeArea + Column + Expanded. Scroll zone = headline through add-item row. Bottom bar = tier banner + CTA, fixed below Expanded (not Spacer). No visible back button on this screen: iOS swipe-back gesture only.
Header
Rule Headline
DM Serif Display, 20px, letter-spacing:-0.3px, colorInk. "Here's what nudgii found." One serif moment on this screen (the headline only).
Note Subtitle
DM Sans italic, 11px, colorMid. "Tap a card to review its tasks. Adjust anything later." Action-oriented guidance, lowers perfectionism.
Category filter
Rule Chip style
Same as S-03. Tinted bg (13% opacity) + category color text. Active = 1px solid categoryColor outline.

"All" active: rgba(155,127,212,0.12) bg, colorCta text, 1px solid colorCta.
"All" inactive: rgba(26,22,18,0.07) bg, colorMid text, no outline.

Behaviour: "All" shows all category sections. Tapping a category shows only that section. Flutter: GestureDetector + AnimatedContainer, 150ms transition.
Tile grid
Rule Layout
2-column CSS grid per category section. gap:9px. Each tile shows: bare colored SVG icon (36px), item name (12px, 600 weight), task count (10px), chevron. Tiles use category tint background at 10% opacity, border-radius:20px.
Rule Remove X
Top-right of each tile, 22px circle, rgba(26,22,18,0.06) bg. Fades to 12% on hover. Flutter: confirm dialog before removing. Removed items can be re-added via "Add another item."
Inline drawer
Rule Drawer panel
White panel, border-radius:20px, 1px solid rgba(26,22,18,0.09). Spans full grid width (grid-column:1/-1). Opens below the tapped tile. One at a time: opening a drawer closes any other open drawer. Flutter: AnimatedContainer, 250ms ease-in-out.
Component Task rows
Category-colored dot on first task only (subsequent dots transparent). Task name: 10.5px, colorMidAccessible. Frequency cadence pill: 8px, light bg, rounded. Delete X per task row (7px icon).

Pro tasks: 38% opacity, purple "Pro" pill, no delete button. Tapping Pro pill navigates to S-22 paywall.
Note Collapse button
Top-right of drawer: upward chevron + "collapse" text, colorMid, 9px. Tapping collapses drawer and returns chevron on tile to default rotation.
Add item row
Rule Normal state
Dashed border (1.5px dashed), border-radius:16px. 28px circle icon with + sign. "Add another item" (11px, 500 weight) + "Search any appliance, vehicle or plant" (9px hint). Spans full grid width.
State Locked (at limit)
When at 10-item free tier limit: lock icon replaces +, "Free plan limit reached" replaces title, "Go Pro" link on right. Dashed border fades to rgba(26,22,18,0.08). Not tappable.
Free tier banner
Rule Pip dots + count
Always visible in sticky bottom bar, above CTA. 10 pip dots (7px circles): filled = colorCta, empty = rgba(26,22,18,0.12). Count text: "X of 10 free items" (9px). "Go Pro" link on right.

Warning state (8+ items): filled pips switch to colorAmber #854F0B.
Full state (10 items): all pips amber, count text in colorAmber.
CTA
Rule "Looks good" button
Full-width pill, colorCta bg, colorCtaFg text, 13px 500 weight. box-shadow:0 4px 18px rgba(155,127,212,0.35). Arrow entity in text. Proceeds to S-05 AHA + celebration. Sticky at bottom, never scrolls.
Navigation
Note Back gesture
iOS swipe-back gesture only. No visible back button on this screen. Swiping back returns to S-02 or S-03 (whichever path the user took).
