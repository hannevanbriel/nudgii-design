Why it matters
Pressure below 1 bar means the boiler won't fire. Above 2.5 bar risks the pressure relief valve. A quick gauge check takes 10 seconds.
How to do it
1
Find the pressure gauge on the front panel
2
Check it reads between 1.0 and 1.5 bar
3
If low, use the filling loop valve to top up slowly

Task expand: full depth inline
Quick info row (since last, times done, cost). "Why it matters" warm card. "How to do it" numbered steps (no connecting lines). Task-level history (last 3 completions with dates + context). Skip count shown as subtle text, not timeline rows. Snooze history never shown. Skip (with return date) + Snooze + Done. Ask nudgii bar.
History: two levels
Task level (inside expand): last 3 completions for THIS task, with date + who/how. Skip count as subtle apricot text ("skipped once this year"), not individual rows. Snooze never shown.
Item level (section below tasks): recent activity across ALL tasks. "All 12 ›" link opens full history. Most recent first. Each row clickable for detail (date, notes, photo).
Skip logic
Button always shows return date. Yearly: pushed 1 month ("Skip · back Nov 2026"). 6-monthly: pushed full interval ("Skip · back Oct 15"). Monthly: pushed full interval ("Skip · back Jun 1").
Done: in-place transformation
