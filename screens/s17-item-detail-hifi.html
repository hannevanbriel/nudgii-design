Structure
Hero + photo area
Category wash bg, 56px icon circle, "Tap to add photo" placeholder. Back/edit/more nav pills float over the hero. Photo replaces placeholder when added.
Stats row (3 boxes)
Owned duration, completions count (sage), coverage remaining (apricot highlight). Coverage box is tappable for warranty detail tooltip.
Status strip
"1 task waiting for you" (not "1 task overdue"). Apricot dot. Next due date right-aligned. Urgency language per CLAUDE.md rules.
Care timeline New
Mini horizontal timeline showing the rhythm of care. Category-colored dots for past completions. Plum dot for next due. Shows the user their care pattern at a glance.
Task cards (inline expand)
Same 40px icon circles as Today tab. Tap to expand: "Why it matters" in category wash, last completed, and actions (Skip/Snooze/Done). "Details ›" link opens S-13 full task page. All icons use category color, never apricot for urgency.
Skip button shows return date
"Skip · back Oct 28" per task-interactions.html. Frequency-aware: ≤1 month pushes by full interval, >1 month pushes by 1 month only. Yearly tasks can't be skipped for a year.
History (clickable rows)
Each completion is tappable (right arrow indicator). Opens completion detail: date, who did it, notes, photo. Sage check icons. Most recent first.
Details (editable)
Brand, model, purchase date, warranty, last service, source. Edit link top-right. Warranty date in apricot when approaching expiry. Dates from AI scan or user input.
Hybrid navigation pattern
Tasks expand inline here for quick actions. "Details ›" opens S-13 for full depth (how-to steps, AI chat, full history). From Today/Plan tab, tapping a task goes directly to S-13.

v1.0 · April 2026 Initial
Initial hi-fi. Replaces S-17 prototype. Key changes from prototype: DM Serif Display + DM Sans (was Noto Serif + Manrope). tokens.css integration. Status strip uses "waiting for you" not "needs attention" per urgency language rules. Mini care timeline added (review feedback #3). All task icons use category color (feedback #6). Task rows match Today tab pattern (feedback #2). History entries are clickable with arrow indicator (feedback #4). Edit button prominent in Details section header (feedback #5). Stats row keeps "X completed" count, not progress fraction (feedback #1). Flutter: ScrollView with SliverAppBar for hero collapse. Task expand uses AnimatedContainer with max-height. Timeline is a CustomPainter with category-colored circles.
