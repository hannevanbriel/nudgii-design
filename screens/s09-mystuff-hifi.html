Layout + shell
Rule Shell pattern
SafeArea + Column + Expanded. Nav bar (84px) fixed at bottom, never scrolls. Content zone: inset: 54px 0 84px 0. Free state adds Go Pro meter (36px) pinned above nav bar, reducing effective scroll area. Padding: 16px horizontal on items list. Scrollbar hidden.
Rule Nav bar
Solid var(--surface) background, no glassmorphism. Active tab: plum icon + plum label, no background shape. Inactive: var(--hint). Tab order: Today, Plan, My stuff, Profile. My stuff = third tab. Icon: shopping bag (Phosphor Bag or equivalent briefcase/bag outline). border-radius: 20px, subtle upward shadow.
Rule FAB
46x46px, border-radius: 14px, plum bg. Plus icon 17px, rotates 45 degrees to X on open. box-shadow: 0 6px 20px rgba(155,127,212,0.45). Position: right: 24px; bottom: 18px, z-index 45, margin-bottom: 4px.
Screen title
Title "My stuff"
Noto Serif, 24px, weight 400, letter-spacing: -0.02em. Note the lowercase 's' in "stuff". Sentence case everywhere. Title row is a flex row (align-items: baseline; justify-content: space-between). Right side: item and task count in Manrope 11px var(--mid) — e.g. "7 items · 24 tasks". Count updates dynamically from item and task queries. Title sits above filter chips with 10px top padding from status bar.
Filter chips
Rule Category filter strip
Horizontally scrollable row, no wrap. "All" always first and active by default. Category pills follow: Home, Garden, Vehicle, Subscriptions, Pets. Active state: var(--ink) background, var(--plum-fg) text, no border, weight 600 — dark ink filled pill. Inactive: transparent bg, ink text, 1px rgba(49,51,44,0.12) border. Chip padding: 5px 12px, 100px radius. No second line ever.
Item card anatomy
Card Layout
Background: var(--surface-low), 12px radius, 14px 16px padding. Each card is its own block with 8px gap between cards. Never border-bottom separators, never multiple items in one container.

Left: 40px round icon circle, category wash bg, category icon SVG 18px.

Right body (flex 1): Header row (justify-content: space-between) — item name 14px 500 ink on the LEFT, status badge text on the RIGHT. Meta line (10px mid, "Category · X tasks"). Functional info line (Noto Serif italic 10px mid, "Last done / Next / Renews"). Mini progress bar (3px height).
Card Mini progress bar
3px height, full card width, border-radius: 100px, overflow: hidden. Track: var(--surface-high). Fill: var(--sage) for on-track items, var(--apricot) for items with overdue tasks, category color for seasonal items. Represents tasks completed this maintenance cycle as a percentage. No label or number.
Status badge variants
Status On track
Text only, no background. Color: var(--sage) (#3B6D11). Font: 10px 500. Shown when all tasks for this item are on schedule with no overdue tasks. Progress bar fill: sage.
Status 1 overdue (or N overdue)
Text only, no background. Color: var(--apricot). Font: 10px 600. Shown when one or more tasks for this item are past due. Progress bar fill: apricot instead of sage. Warm warning, not red anxiety.
Status Seasonal
Text only, no background. Color: category color (e.g. var(--cg-bg) for garden). Font: 10px 500. Shown when item has no tasks due right now because it's in an off-season window (e.g. garden in winter). Progress bar fill: category color. Tapping goes to item detail where seasonal schedule is explained.
Status Paused (free plan, Pro item)
Item card opacity: 0.60. "Pro" pill in plum (see spec below). Status text: "Paused" in hint color. No reminders sent, no push for this item. Progress bar at 0%. Tapping the card opens an upgrade bottom sheet explaining the limit and offering to set one of the 10 active items as paused instead. Item is never hidden or deleted.
Spec Pro pill
8px font, weight 700, uppercase, letter-spacing 0.04em. Padding: 2px 6px, 4px radius. Background: var(--plum-wash). Text: var(--plum). Border: 1px solid rgba(155,127,212,0.25). Sits inline in the item name row, right of the name. Flutter: Badge widget or Container with decoration.
Add item card
Card Dashed border
Full-width card, margin 0 16px. Border: 1.5px dashed var(--plum), 12px radius. No background fill (transparent). Centered content: 32px plum-wash circle with plus icon (14px plum) + "Add an item" (14px 500 plum) + "or scan something" (10px hint). Padding 20px vertical. Tap: opens FAB action sheet or adds item flow. Active state: plum-wash background tint.
Go Pro meter
Spec Free state meter
Position: Fixed strip between scroll content and nav bar. position: absolute; bottom: 84px (above nav). Full width. Surface-low background, surface-high top border.

Content: Left: "7 of 10 free" (11px 500 ink). Center: 10-dot meter (6px circles, 3px gap) — plum filled dots for active items, surface-high empty dots for remaining slots. Right: "Go Pro" link (11px 500 plum).

Visibility: Free plan only. Completely hidden on Pro state. No animation on hide/show.

Flutter: Column with fixed bottom widget outside Expanded scroll area. Not inside SafeArea. Update dot count from activeItemCount provider.
Functional info line
Copy Last done / Next / Renews
Noto Serif italic, 10px, var(--mid). One line only. Shows actionable info derived from task history and upcoming schedule. Three variants:

Last done: "Last done: [task name], [relative time]" — for items where the most recent signal is a completed task. Source: lastCompletedTask from Supabase.

Next: "Next: [task name], [day date]" — for items with an upcoming task soon. Source: nextTask.dueDate.

Renews: "Renews: [Month DD]" — for subscription items. Source: subscriptionRenewalDate.

Seasonal: "[Season] [action] window now open" — for seasonal items. Source: seasonal schedule engine.

Per item: Boiler "Last done: annual service, 6 months ago", Dishwasher "Last done: salt refilled, 2 months ago", Volkswagen Golf "Next: check tyre pressure, Wed 26", Netflix "Renews: May 12", Olive tree "Spring pruning window now open", Cat "Next: annual vet check, Jun 3", Robotic lawn mower "Last done: winter storage, Nov 23".
States
State 1 Free (default)
When: User has items and is on the free plan with 7 active items. 1 paused item (item 11+) shown muted. Go Pro meter visible above nav bar. Items 1-7 are active, item 7 (paused Pro item) is muted at 0.60 opacity with Pro pill and no reminders.
State 2 Pro
When: User is on Pro plan. No item count limit. All items fully active, no paused state, no Pro pills. Go Pro meter completely hidden. Otherwise identical layout and item card anatomy.
State 3 Empty
When: No items added yet (new user or all items deleted). No filter chips shown. Centered content: shopping bag icon (64px surface-low circle), Noto Serif "No items yet." (22px), Manrope body copy (13px mid), plum pill CTA "Add my first item". Copy consistent with nudgii voice: direct, not salesy.
Item tap behavior
Tap Goes to S-17 item detail
Tapping any active item card navigates to S-17 item detail. Transition: slide from right (300ms ease-in-out). Tapping a paused item on free plan opens upgrade bottom sheet instead (not item detail). Add item card opens FAB action sheet or scanner flow.
Dark mode
Rule Dark tokens
Surface: #1C1B18 (warm dark). Plum lightens to #B8A0E8, apricot to #F0B890. Category washes at 15% opacity (was 14% light). Text primary: #F0EDE6. Mid: #A8A29A. Nav bar: surface-low (#24231F). Surface-high: #2E2D28. Item cards: surface-low background. No blue or cool-green tints. Toggle with [data-theme="dark"].
Flutter notes
Flutter Implementation
List: SliverList inside CustomScrollView for performance with many items. Each item card: GestureDetector wrapping Container with category wash background. No Card widget (adds unwanted elevation).

Progress bar: ClipRRect + LinearProgressIndicator, height 3px, borderRadius: BorderRadius.circular(100). Color from task completion ratio.

Go Pro meter: Fixed Column child below Expanded scroll area, outside SafeArea. Visibility: Consumer watching plan status from RevenueCat provider. Dot count: activeItemCount from Supabase items query.

Filter chips: SingleChildScrollView(scrollDirection: Axis.horizontal) with Row of FilterChip. State: local StatefulWidget, notify parent for filter query.

Paused item: Opacity(opacity: 0.60) wrapping item card. Tap opens showModalBottomSheet with upgrade content instead of navigating to item detail.

v1.1 · 2026-04-02 · BREAKING
Aligned with Precision Warmth prototype. Four breaking changes from v1.0:

Item and task count in title: "7 items · 24 tasks" added as a small right-aligned secondary text (11px, var(--mid)) next to the "My stuff" serif headline. Screen title row is now a flex row with space-between.

Status badge moved to right side of card: Previously inline next to item name (e.g. "Boiler On track"). Now right-aligned in the header row using justify-content: space-between on .ic-head. Item name is left, status badge is right. The badge was never a background pill, it remains text-only — only its layout position changed.

Personality line replaced with functional info: The Noto Serif italic "personality" line (e.g. "Reliable since 2021") is replaced with a functional info line (class .ic-functional, same styling: Noto Serif italic, 10px, var(--mid)). Content is now actionable: "Last done: [task], [time ago]", "Next: [task], [date]", or "Renews: [date]" depending on item type and status. Personality lines archived — content comes from task history and schedule data, not a static string.

Filter chip active state changed: Active "All" chip changed from plum outline (border + text + wash bg) to dark ink filled pill (background var(--ink), text var(--plum-fg), no border). Inactive chips: transparent bg, ink text, 1px ink-12 border. Matches the prototype's stronger visual anchor for the active filter.
Flutter notes:
Title row: Row(mainAxisAlignment: MainAxisAlignment.spaceBetween) with serif heading left and Manrope 11px mid text right. Count derived from item query result count + summed task count.
Status badge position: item card header uses Row(mainAxisAlignment: MainAxisAlignment.spaceBetween). Badge text widget stays the same (no background Container), only layout changes.
Functional info line: source from lastCompletedTask?.completedAt (for "Last done") or nextTask?.dueDate (for "Next") or subscriptionRenewalDate (for "Renews"). Format relative dates with timeago package for completed tasks, absolute day names for upcoming tasks.
Filter chip active state: use FilterChip with selectedColor: colorInk, labelStyle: TextStyle(color: colorCtaFg), side: BorderSide.none when selected. Inactive: backgroundColor: Colors.transparent, side: BorderSide(color: colorInk.withOpacity(0.12)), labelStyle: TextStyle(color: colorInk).
v1.0 · 2026-04-02 · ADDITIVE
Initial hi-fi for My stuff tab. New file. Three states: Free (default), Pro, Empty.

Item cards: Full-width surface-low cards with 8px gap. Each card: 40px category icon circle + item name with status badge + "Category · X tasks" meta + Noto Serif italic personality line + 3px mini progress bar. No border-bottom separators. No grouped containers.

Status badges: Text-only, no background. On track: sage. Overdue: apricot. Seasonal: category color. Paused: hint text + Pro pill. Progress bar fill matches status color.

Go Pro meter: Fixed strip above nav bar (free state only). "7 of 10 free" label + 10-dot meter (7 plum filled, 3 empty) + "Go Pro" link. Completely hidden on Pro state.

Add item card: Dashed plum border, centered plus + "Add an item" + "or scan something" sub. Transparent background, 12px radius. Sits after all items in the list.

Paused item (free): Item 7 (robotic lawn mower) shown at 0.60 opacity with Pro pill, no reminders, progress at 0%. Tapping opens upgrade sheet.

Empty state: Centered icon + Noto Serif "No items yet." + body copy + plum pill CTA. No filter chips shown when empty.

Dark mode: Full token set applied via [data-theme="dark"]. Warm dark surfaces, brightened plum and apricot. No blue tints.
Flutter notes:
Mockup base: 390x844 (iPhone 15). SafeArea: yes on content, not on Go Pro meter strip. Scroll: CustomScrollView + SliverList for item list. Bottom zone: nav bar 84px fixed, Go Pro meter 36px fixed above nav (free only). Item cards: GestureDetector + Container, no Card widget. Progress bar: ClipRRect + LinearProgressIndicator at 3px height. Paused item: Opacity(0.60) wrapper. Filter chips: SingleChildScrollView(horizontal). Plan status: RevenueCat.purchaserInfo to derive free/pro. Go Pro meter visibility: conditional on plan status. Add item card: dashed border via ShapeDecoration + DashedBorder or custom painter.
