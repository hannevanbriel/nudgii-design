AHA + celebration + sign-in

Two-state screen combining AHA schedule reveal + plum celebration + sign-in bottom sheet. Replaces separate S-06 AHA, S-07 Celebration, and S-08 SSO screens. The user taps "That's it. I'm done." and the screen transitions from cream to plum with a spring-animated celebration sequence. After 1.5s the sign-in bottom sheet slides up.

Transition + motion
Decision Two-state transition
Tapping "That's it. I'm done." triggers a spring transition from cream to plum. Background crossfades via AnimatedContainer (400ms ease-in-out). State 1 content fades out (300ms). State 2 celebration content animates in with staggered timing. This is the emotional peak of onboarding: the user sees their schedule is real.
Rule Celebration sequence timing
150ms: Background transitions cream to plum, State 1 fades out.
250ms: Confetti burst begins, pieces staggered at 30ms intervals.
400ms: ii mark springs in (light-on-dark variant, cubic-bezier(0.34,1.56,0.64,1) 500ms).
550ms: "you're in" eyebrow fades in.
650ms: Headline slides up + fades in (350ms ease-out).
800ms: Subtitle fades in.
900ms: First reminder pill fades in.
1500ms: Sign-in bottom sheet slides up with scrim.
Flutter: use Future.delayed chain or AnimationController with staggered intervals.
Rule Progress bar
Full-width, edge-to-edge, outside SafeArea. State 1: 80% fill on light track (rgba(26,22,18,0.08)). State 2: animates to 85%, track switches to rgba(255,255,255,0.08) for dark background. Fill color remains colorCta #9B7FD4 in both states. Flutter: AnimatedContainer for track color transition.
State 1: Schedule reveal
Eyebrow + headline
Eyebrow: "Your schedule is ready" in colorCta #9B7FD4, 10px, uppercase, 1.5px letter-spacing. Headline: DM Serif Display 24px, colorInk. Subline: 13px italic DM Sans, colorMid. The copy reinforces the AHA moment: nudgii figured it out, the user just confirmed what they have.
Rule Task cards
White cards on cream, 6px gap. Each card: 10px border-radius, 0.5px border rgba(26,22,18,0.08), box-shadow 0 1px 3px rgba(26,22,18,0.04). Category icon sits directly on card — no icon container. Icon stroke = category icon token (e.g. Home: catHomeIc #7A5A50). Task name 13px/500, subtitle 10px colorMidAccessible. Date label: colorInk at 55% opacity, no background. Tasks shown are personalized based on what the user added in S-02/S-03.
Rule CTA button
"That's it. I'm done." in colorCta fill, colorCtaFg text, pill radius, 50px height. Shadow: 0 2px 12px rgba(155,127,212,0.28). Pressed: #7D62B3 bg, scale 0.97, 100ms. Copy is deliberately casual and final: the user is done with input, nudgii takes over.
State 2: Celebration
Decision Confetti
40 pieces in v10.5 category colors (#7A5A50, #9B7FD4, #607870, #4A6A82, #C4A8E8, #6A5085, #F5F0E8) at 55-95% opacity. Mix of circles and rectangles. Fall from top with staggered start (30ms per piece), sway animation, 2.5-5s fall duration. Flutter: consider confetti_widget package or custom painter with AnimationController.
Rule ii mark (light-on-dark)
Light-on-dark variant: left pill #F5F0E8 (cream), right pill #9B7FD4 (CTA), dot #C4A8E8 (colorPlumPale — replaces colorApricot). 48x54px centered. Spring animation: cubic-bezier(0.34,1.56,0.64,1) 500ms, scales from 0.3 to 1.0 with overshoot. This is the only theatrical motion in the app.
Celebration copy
Eyebrow: "you're in" (lowercase, rgba(245,240,232,0.40)). Headline: personalized italic e.g. "Your boiler is relieved." in DM Serif Display italic 26px, colorPlumPale #C4A8E8 (replaces colorApricot). Subtitle: "We'll remind you when it matters." in italic 12px, rgba(245,240,232,0.50). First reminder pill shows the nearest upcoming date; date accent in colorCta #9B7FD4 (replaces colorApricot).
State 3: Sign-in sheet
Rule Bottom sheet
White surface, border-radius: 20px 20px 0 0, shadow 0 -8px 30px rgba(0,0,0,0.20). Handle bar: 32x3px, rgba(26,22,18,0.12). Slides up with 350ms ease-out animation. Scrim behind: rgba(0,0,0,0.25). Celebration content remains visible but dimmed (opacity 0.5). Flutter: showModalBottomSheet with custom shape and barrier color.
Sign-in content
ii mark (dark-on-light, 20x22px) above "Save your schedule" (DM Serif Display 22px). Subtitle: "Your data stays yours. We never share it." (italic, colorMid). Privacy-first framing. Apple button: colorInk bg, white text, 50px. Google button: bordered, 48px. "Not now" text link below (13px, colorMid).
Decision Navigation from sign-in
All three paths (Apple, Google, "Not now") navigate to S-06 Push permission. Signing in saves the schedule to their account. "Not now" stores the schedule locally on device. The push permission screen follows regardless because push notifications are the core delivery mechanism. Signed-in users see personalized greeting on dashboard, skipped users see "Good morning, there."

v1.0 · 2026-03-26 · NEW
Two-state screen combining AHA reveal + celebration + sign-in. Replaces separate S-06 AHA, S-07 Celebration, and S-08 SSO screens. Reduces onboarding from 8 screens to 6 by merging three related moments into one continuous transition.
State 1 (cream): Schedule reveal with personalized task list based on user input from S-02/S-03. Eyebrow "Your schedule is ready" + headline + 4 task cards. CTA "That's it. I'm done." triggers transition.
State 2 (plum): Spring transition to plum background. Confetti burst (40 pieces, category colors, staggered 30ms). ii mark springs in with cubic-bezier(0.34,1.56,0.64,1). Celebration copy personalized to first item. First reminder pill with nearest date.
Sign-in bottom sheet: Slides up over plum after 1.5s delay. White sheet with handle bar, ii mark, "Save your schedule" headline. Apple + Google SSO buttons + "Not now" link. All paths navigate to S-06 Push.
Progress bar: 80% in State 1 (cream), animates to 85% in State 2 (plum). Track color transitions from light to dark.
Standard Flutter implementation block · S-05

SafeArea:       top=false, progress bar outside SafeArea
Scroll:         SingleChildScrollView for State 1 content zone only
Bottom zone:    State 1: sticky CTA, fixed Column at bottom. State 2: none (centered content). State 3: showModalBottomSheet

Scaling:        All visual elements x (screenWidth/390).clamp(0.9, 1.15)
Special:        AnimatedContainer for cream-to-plum bg transition (400ms). Transform.scale with spring curve for ii mark. showModalBottomSheet for sign-in (350ms slide-up, isDismissible: true). Confetti: custom painter or confetti_widget package. Staggered Future.delayed chain for celebration sequence.
