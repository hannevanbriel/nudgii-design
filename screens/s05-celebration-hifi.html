State 1: Rich AHA layout
Decision Rich AHA replaces Plan-style task list
The Plan-style flat task list was informational but not emotional. The Rich AHA layout creates a stronger moment of realization: nudgii understood your stuff and built a plan. The ii mark anchors the screen as a nudgii moment. Stats prove the work happened. The first nudge card shows depth (why + how). The month bar shows breadth (your whole year).
Rule ii mark (dark-on-light, centered)
32x36px, centered. Dark-on-light variant: left pill colorCta #9B7FD4, right pill colorPlum #2D1F4A, dot colorPlumPale #C4A8E8. Sits at the top of the scroll content, 28px top margin. This is the brand anchor. Flutter: SvgPicture.asset centered in a Row(mainAxisAlignment: center).
Headline + subline
Headline: "You said what. We figured out when and why." DM Serif Display 22px, centered, colorInk. Two sentences, line break between them. The copy mirrors the product promise: user provides items, nudgii provides the schedule and the reasoning. Subline: "Your maintenance schedule is ready." 12px italic DM Sans, colorMid.
Rule Stats row
Three pills: items, tasks, months. Each pill: surface bg, 0.5px border, pill radius, 11px DM Sans 500. Number in colorCta bold. These are personalized: actual counts from the user's input in S-03/S-04. "Months" = the time range covered by all scheduled tasks. Flutter: Row(mainAxisAlignment: center) with Container pills.
Rule "Your first nudge" card
Section label: 9px uppercase, colorCta, 1.5px letter-spacing. Card: 14px radius, surface bg, 0.5px border. Layout: 44px icon circle (category wash bg) + task name (14px/500) + item/category meta (10px mid) + countdown badge (apricot text on apricot-wash bg). Below divider: "Why it matters" section (apricot label, 11px body) and "How to do it" section (CTA purple label, 11px body). Always pre-expanded, no chevron. This is the AHA payoff: nudgii doesn't just remind, it explains.
Rule "Year at a glance" month bar
6 month columns. Each: 40px track height, 4px radius, ink-06 bg. Fill height proportional to task count (max month = 100%). Fill color = dominant category color for that month at 35% opacity. Month label: 8px uppercase, colorMid. Count: 7px, colorHint. Bars grow from bottom (flex-direction: column-reverse). Flutter: Row of Column widgets, heights calculated as (monthTasks / maxTasks) * 40.
Rule CTA button
"That's it. I'm done." in colorCta fill, colorCtaFg text, pill radius, 50px height. Shadow: 0 2px 12px rgba(155,127,212,0.28). Sticky at bottom via flex layout. Copy is deliberately casual and final: the user acknowledges nudgii's work. Triggers the cream-to-plum celebration transition.
Transition + motion
Decision Two-state transition
Tapping "That's it. I'm done." triggers a spring transition from cream to plum. Background crossfades via AnimatedContainer (400ms ease-in-out). State 1 content fades out (300ms). State 2 celebration content animates in with staggered timing. This is the emotional peak of onboarding.
Rule Celebration sequence timing
150ms: Background transitions cream to plum, State 1 fades out.
250ms: Confetti burst begins, pieces staggered at 30ms intervals.
400ms: ii mark springs in (light-on-dark variant, cubic-bezier(0.34,1.56,0.64,1) 500ms).
550ms: "you're in" eyebrow fades in.
650ms: Headline slides up + fades in (350ms ease-out).
800ms: Subtitle fades in.
900ms: First reminder pill fades in.
1500ms: Sign-in bottom sheet slides up with scrim.
Flutter: use Future.delayed chain or AnimationController with staggered intervals.
Rule Progress bar
Full-width, edge-to-edge, outside SafeArea. State 1: 80% fill on light track (rgba(26,22,18,0.08)). State 2: animates to 85%, track switches to rgba(255,255,255,0.08) for dark background. Fill color remains colorCta #9B7FD4 in both states.
State 2: Celebration
Decision Confetti
40 pieces in v10.5 category colors at 55-95% opacity. Mix of circles and rectangles. Fall from top with staggered start (30ms per piece), sway animation, 2.5-5s fall duration. Flutter: consider confetti_widget package or custom painter with AnimationController.
Celebration copy
Eyebrow: "you're in" (lowercase, rgba(245,240,232,0.40)). Headline: personalized italic e.g. "Your boiler is relieved." in DM Serif Display italic 26px, colorPlumPale #C4A8E8. Subtitle: "We'll remind you when it matters." in italic 12px. First reminder pill shows the nearest upcoming date; date accent in colorCta #9B7FD4.
State 3: Sign-in sheet
Rule Bottom sheet
White surface, border-radius: 20px 20px 0 0, shadow 0 -8px 30px rgba(0,0,0,0.20). Handle bar: 32x3px. Slides up with 350ms ease-out. Scrim: rgba(0,0,0,0.25). Celebration content dimmed (opacity 0.5). Flutter: showModalBottomSheet with custom shape.
Decision Navigation from sign-in
All three paths (Apple, Google, "Not now") navigate to S-06 Push permission. Signing in saves the schedule to their account. "Not now" stores locally. Push permission follows regardless. Signed-in users see personalized greeting on dashboard, skipped users see "Good morning, there."
