S-01 · what changed
↑
Task rows replace item rows. Shows nudgii's pre-loaded knowledge before the user does anything — cadence is visible immediately.
↑
2 CTAs instead of 3. "Tell me" (primary) + "Show me" (secondary). Scan moves to a subtle sub-link — visible to power users, not competing for attention.
★
Selection is optional context — CTAs are always active. Selecting items carries context into S-02 ("Winter tyres noted") or highlights categories in S-03.
S-02 · new unified input
N
Voice + text + camera in one bar. Modality is the user's in-the-moment choice — not a separate path decision made upfront on S-01.
★
Context pill shows S-01 selection — "Winter tyres noted." Makes the tap on S-01 feel meaningful, not decorative.
★
Privacy note: "voice stays on your device" — one line, small, but builds trust for users who hesitate on voice permissions.
S-03 · browse + search merged
N
Search filters browse in real-time. No separate search screen. Sara-persona (power user) taps "Show me", uses the search bar immediately — same destination, zero friction.
★
Task count badge per item ("3 reminders") communicates depth without listing tasks. User understands value before committing.
★
Running CTA count — "Continue with 2 items" updates live. Creates momentum, no dead-end state when 0 items selected.
S-05 · review updates
↑
Scan icon per item card (top-right). Tap → camera → Claude API returns brand + model → tasks update with more precision. Optional, never required.
★
Cadence on every task ("every 6–8 wks", "before Nov · BE"). Reinforces the core value: nudgii knows timing, not just task names.
★
Status dots: terra = overdue, apricot-dk = due soon, mid = neutral. Visual hierarchy without text labels cluttering the task names.
S-07a · new · AHA before auth
N
Value before the wall. User sees real upcoming tasks with real dates before any auth screen. The schedule is already built — they're just saving it.
N
"Save my schedule" reframes auth. Not "create an account" — they're protecting something they already own. Proven conversion pattern (Duolingo, Notion, Calm).
★
Value callout below cards: "You told us what. nudgii figured out when." The proposition is proven by the cards above it — not just stated.
