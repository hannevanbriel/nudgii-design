Browse categories

Rule Progress bar
Starts at 40% (step 2 of 5). Grows by 8% per selected item, clamped at 80%. Animates on every toggle via AnimatedContainer. Outside SafeArea, full-width edge-to-edge.
Category chips
Rule New chip style — no dots
Category chips use tinted background + category color text. No dot indicators. Each chip background = categoryColor at 13% opacity. Text = full category color.

Active category: adds 1px solid categoryColor outline.
"All" inactive: rgba(26,22,18,0.07) bg, colorMid text, no outline.
"All" active: rgba(155,127,212,0.12) bg, 1px solid colorCta, colorCta text.

Flutter: GestureDetector + AnimatedContainer. No FilterChip widget (insufficient animation control).
Tile grid
Rule Overview mode — horizontal rows
One section per category. Each section: category label + horizontal SingleChildScrollView row. EdgeInsets.symmetric(vertical:4, horizontal:16) — vertical padding prevents selected border clip.

Row ends with a faint › peek indicator (rgba(26,22,18,0.20)).

Swipe hint (first load only): Home row scrolls 52px right, pauses 520ms, snaps back. hintDone flag prevents repeat.
Rule Category mode — 3-column grid
Tapping a category chip switches to GridView, 3 columns, gap:7px. Selected items sort to the top. No overflow tile or "something else" row in this view.
Rule Tile widget
88px wide, 14px radius. position: relative for check circle overlay.

Unselected: solid pastel bg (Home #D2BEB6, Vehicle #C0CDD8, Garden #C0CEC9, Subs #D0C6DC). No border, no indicator.
Selected: deepened bg (Home #C4A89E, Vehicle #A8B8C4, Garden #A8B8B4, Subs #BCB0CC) + filled circle top-right (16px, category icon color bg, white tick 9px).

Icon: Phosphor outline, 26px, category icon color, no inner wrapper.
Label: DM Sans 9.5px, 500 weight, colorInk, centered, max 2 lines.

Hover: scale 1.03. Press: scale 0.93. Haptic: HapticFeedback.selectionClick().
Selected pill
Appears when selectedIds.isNotEmpty. colorCta filled pill, white-tinted count circle on left. Tapping shows only selected items in a 3-col grid (temporary filter view).
CTA progression
Rule Copy states
0 items: rgba(26,22,18,0.08) bg, colorMid text, "Tick something first", not tappable.
1 item: Active CTA purple, "Good start, continue" + count badge.
2 items: "Nice, keep going or continue".
3+ items: "Looking good, continue".

Copy transition: fade out 180ms, swap copy, fade in. Count badge updates alongside. All via AnimatedContainer 200ms.
State model
Riverpod state
CatFilter enum: all, home, vehicle, garden, subs.
S03State holds: activeFilter, searchQuery (lowercase), Set<String> selectedIds.
All rendering derives from state. Pre-selected IDs passed from S-01 as initialSelectedIds (empty in v2 — S-01 no longer has interactive selection).

Flutter implementation block · S-03 v2.0
Mockup base:      390px width (iPhone 15)
SafeArea:         top=false — progress bar bleeds full width above SafeArea
Scroll:           Expanded + SingleChildScrollView for content zone only
Bottom zone:      Fixed Column at bottom (not Spacer). Hairline + CTA button.
Progress bar:     Full width, height 3px. Width = (0.40 + selectedCount * 0.08).clamp(0.40, 0.80) * screenWidth. AnimatedContainer 500ms ease.
Scaling:          All visual elements x (screenWidth/390).clamp(0.9, 1.15)
State:            Riverpod. S03State(activeFilter: CatFilter, searchQuery: String, selectedIds: Set<String>)
Category chips:   GestureDetector + AnimatedContainer. No FilterChip widget. Smooth 150ms transition.
Tile rows:        SingleChildScrollView(scrollDirection: Axis.horizontal). EdgeInsets.symmetric(vertical:4, horizontal:16).
Tile grid:        SliverGrid with 3-column delegate when activeFilter != all.
Tile widget:      GestureDetector + AnimatedContainer. On tap: HapticFeedback.selectionClick().
Swipe hint:       ScrollController on Home row. Post-frame callback: scrollTo(52, duration:400ms), then scrollTo(0, duration:300ms) after 520ms delay. hintDone flag in state.
CTA copy:         AnimatedSwitcher 180ms on Text widget. Fade transition.
Data:             Query item_types joined with categories. Group by categories.slug. Sort by sort_order. icon_key via ItemIcons.fromString().
Navigation:       CTA onTap (selectedIds.isNotEmpty): push to S-04 Review passing selectedIds.
