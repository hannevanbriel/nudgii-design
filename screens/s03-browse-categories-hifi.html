Structure
Rule Progress bar
Starts at 40% (step 2 of 5). Grows by 8% per selected item, clamped at 80%. Animates on every toggle.
Selection pattern
Decision Entity cards replace tile grid
Items shown as entity cards matching My Stuff: 44px icon circle, DM Serif name, category wash bg. Selection state: filled check circle (24px, category color bg, white tick). Unselected: empty circle with thin border. Card wash stays the same in both states. Visual continuity with S-04 Review and dashboard.
Rule Category filter chips
"All" active by default (plum fill). Category chips: neutral bg until tapped. Tapping a category filters the list to only that category's items. Single selection: tapping a new chip deselects the previous one.
Rule Search bar
Filters items by name as user types. Phosphor MagnifyingGlass icon, 11.5px italic placeholder. Positioned above chips, below header.
CTA progression
Rule Copy states
0 items: Disabled, "Tick something first".
1 item: Active, "Good start, continue [1]".
2 items: "Nice, keep going or continue [2]".
3+ items: "Looking good, continue [N]".
180ms fade transition on copy swap.
Dark mode
Rule Dark tokens
Surface #1C1B18. Plum #B8A0E8. Category washes use lightened hues at 12% opacity. Icon circles: rgba(255,255,255,0.08). Search bar: rgba(255,255,255,0.04) bg. Check circle: category color bg unchanged (already bright enough). CTA: #B8A0E8 bg.

Flutter implementation block · S-03 v3.0
Mockup base:      390px width (iPhone 15)
SafeArea:         top=false, progress bar outside SafeArea
Scroll:           Expanded + SingleChildScrollView for content zone
Bottom zone:      CTA in fixed Column (not Spacer)
Progress bar:     (0.40 + selectedCount * 0.08).clamp(0.40, 0.80)
Scaling:          Text clamp 1.3x. Elements x (screenWidth/390).clamp(0.9, 1.15)
State:            Riverpod. S03State(activeFilter, searchQuery, selectedIds)
Selection:        Tap toggles selectedIds. HapticFeedback.selectionClick().
Theme:            [data-theme] maps to ThemeData.
Navigation:       CTA onTap: push to S-04 Review passing selectedIds.
