1
Progress bar
Single continuous bar · 3px · full-width edge-to-edge. S-01 = 20% fill. Fills left to right across 5 steps: 20% → 40% → 60% → 80% → 100%. Fill color: colorCta #9B7FD4. Track: rgba(26,22,18,0.08).
2
ii mark — locked
20px height · left-aligned. Left pill: ink. Right pill: plum-lt at 45% opacity. Apricot-dk dot. Mirrors the homescreen icon — same three colors, same proportions.
3
Free badge
"Free · no credit card" — sage-lt #EAF3DE background · sage #3B6D11 text · sage dot. Positioned top-right. Communicates "free = positive," not "free = limited."
4
Headline
DM Serif Display 33px · lh 0.97 · ls -0.55px. "maintain itself." in em tag → italic → colorApricotDk #C8784A. One serif moment per screen. Sub: DM Sans 400 italic 12px mid.
5
Preview items — item row style
3 rows using dashboard visual language. White card on cream · category icon chip 28×28px · item name 13px/500 · status 11px. Overdue = terra. Never mini-tile columns, never a shared card wrapper.
