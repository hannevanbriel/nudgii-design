v10.5 changes
v3.0 4-tab bar: Today, Plan, My Stuff, Profile
Replaces the old 3-tab layout (Tasks, Ask, Profile). Plan tab shows month-grouped task list. My Stuff shows item tiles grouped by category. Custom widget only, never BottomNavigationBar. Active tab: colorCta on both icon stroke and label. Inactive: rgba(26,22,18,0.20). No pip dot.
Interaction Expandable task cards (feat pattern)
Tap card header to expand. Reveals: "Why it matters" context box + Done / Snooze / Skip action buttons. Chevron rotates 180 degrees when expanded. Card background: solid category pastel, no border, border-radius: 16px. Flutter: AnimatedCrossFade or AnimatedSize for expand. Done button: colorCta. Snooze: rgba(44,40,36,0.08). Skip: rgba(44,40,36,0.06).
Rule Progress summary ring
44x44px SVG ring, colorCta stroke, 3px width. Centered fraction label (9px 600 weight). Ring: stroke-dasharray="113" (full circumference), stroke-dashoffset = proportion incomplete. Next to ring: attention count (13px 500) + on-track count (11px mid). Right side: week streak badge (number in cta, "week streak" label, cta-tint bg). Flutter: CustomPaint with drawArc.
Note Dynamic cards row
Horizontal scroll, 148px wide cards, 8px gap. Three card types: "Did you know?" (catHomeBg), savings summary (cream tint), seasonal progress (catGardenBg). Cards are dismissible. Content sourced from backend, not hardcoded. Flutter: SingleChildScrollView(scrollDirection: Axis.horizontal).
Rule Overdue badge
Cream text on colorOverdueBadge (#6B4A3A) pill. Format: "12d overdue". Pill: border-radius: 100px; padding: 3px 9px; font-size: 9px; font-weight: 600. Replaces colorTerra text label from previous version. Never red, never terra red.
Navigation
Rule FAB position
FAB sits outside the tab bar pill, bottom-right. position: absolute; right: 20px; bottom: 18px relative to screen. 46x46px, 15px radius (rounded square, not circle). colorCta bg. Shadow: 0 6px 20px rgba(155,127,212,0.45). Plus rotates 45 degrees to X on open using AnimatedRotation.
Rule ii mark avatar
42x42px circle, rgba(44,40,36,0.06) bg. Replaces plain settings icon in greeting row. Two-path SVG: first i (left, faint outline stroke), second i (right, cta-tinted fill). Top-left of greeting row. Flutter: use inline SVG via SvgPicture.string or embed as asset.
Decision Free tier (State 4)
Purple-tinted banner: "You added 13 items. Free plan tracks 10. 3 items are paused." with "Manage" link. Paused tasks: opacity 0.45, "Pro" pill (colorCta text + border). Conversion trigger: when a paused item's task comes due. No auto-delete, no auto-hide, never block data.
