Shell + layout
Rule Greeting personalization
Signed-in users: "Good morning, Martijn." (name from SSO, DM Serif Display 25px, name in italic colorApricot). Skipped users: "Good morning, there." Same styling, generic fallback. Time-aware: morning/afternoon/evening. Flutter: Text.rich() with separate TextSpan for name.
Rule Settings icon
PhosphorIcons.faders, never GearSix. 16px icon in a 34x34px tappable container (meets 44px minimum when including padding). colorMidAccessible #6B6358 stroke. Top-right of greeting row.
Rule Category filter strip
"All" pill always first, always the initial active state. Single scrollable row (flex-wrap:nowrap; overflow-x:auto). Active pill: colorCta border + text, subtle lavender bg. Category pills include 6px colored dot. Flutter: SingleChildScrollView(scrollDirection: Axis.horizontal). Never wraps. Omitted on empty state (nothing to filter).
Task cards
Rule Card anatomy
White card on cream, 8px gap (dashboard uses 8px, onboarding uses 6px). Each card: category icon chip (28x28, category-colored bg, Phosphor outline) + task name (13px DM Sans 500) + item/cadence subtitle (11px italic colorMid) + date label right-aligned. No left border accent lines, ever. Status lives in date label color only.
Note Date label colors
  • Overdue: colorTerra #C4503A
  • Due soon (within 7 days): colorApricotDk #C8784A
  • Normal: colorMid #8A8070
Interaction Swipe to complete
Swipe right to complete, threshold 40%. Gesture hint ("swipe right to complete") shown below first overdue/due-soon task on first visit. Hint disappears after first completion. Flutter: Dismissible with custom confirmDismiss. Done state: AnimatedOpacity to 0.35 + strikethrough, 250ms.
States
Decision Empty state (State 2)
Centered vertically. DM Serif Display 22px headline: "Nothing here yet." Italic subtitle. No category filter strip (nothing to filter). FAB is the primary action path. No additional CTA needed.
Decision Free tier limit (State 3)
Purple-tinted banner explains: "You added 13 items. Free plan tracks 10. 3 items are paused." with "Manage" link. Paused tasks: opacity 0.45, "Pro" pill badge (colorCta text + border). Conversion trigger: when a paused item's task comes due. See CLAUDE.md Section 3 for full tier details.
Decision All done (State 4)
Light celebration on cream (not the plum done overlay, that is for individual task completion via swipe). DM Serif Display 22px italic: "Nothing overdue. Suspicious." Witty, on-brand. Subtitle reassures: "Enjoy the quiet. We'll nudge you when something's coming up." Subtle checkmark next to "0 tasks this week".
Decision Digest nudge (State 5)
Shown to users who skipped sign-in. "Get a weekly summary in your inbox." with "Sign in to enable" CTA link. Opens sign-in bottom sheet (same as S-05). Dismissible via X button. Single most important leading indicator: weekly digest open rate.
Navigation
Rule Tab bar
Custom widget, NOT BottomNavigationBar. Floating pill: white, 22px radius, 0 4px 20px rgba(26,22,18,0.11) shadow. Active tab: colorCta #9B7FD4 on both icon stroke and label. No pip dot. Inactive: rgba(26,22,18,0.20). 14px from screen edge. Three tabs: Tasks, Ask, Profile.
Rule FAB
46x46px, rounded square (15px radius, NOT circle). colorCta background. Plus icon 17px colorCtaFg. Shadow: 0 6px 20px rgba(155,127,212,0.45). Position: bottom-right, adjacent to tab bar. margin-bottom: 4px for elevation. Tapping opens FAB action sheet (S-12). Plus rotates 45 degrees to X using AnimatedRotation.
Changelog
v1.0 · 2026-03-26 · NEW
Initial hi-fi with 5 states. Default (signed in, under 10 items), empty (no items), paused items (free tier limit at 10), all done, and skipped sign-in (with digest nudge card).
Greeting personalization. Signed-in users see name from SSO ("Good morning, Martijn."). Skipped users see "Good morning, there." Name rendered in italic colorApricot.
Category filter strip. "All" pill always first, always active by default. Omitted on empty state. Active state uses purple outline (colorCta border + text, lavender bg).
Task card anatomy. White cards on cream, 8px gap. Category icon chip + task name + item/cadence subtitle + date label. Date color encodes status: terra (overdue), apricot-dk (due soon), mid (normal). No left border accent.
Free tier banner. Purple-tinted card explaining item limit. Paused tasks at opacity 0.45 with "Pro" pill badge. "Manage" link for item selection.
All-done state. Light celebration on cream (not plum overlay). "Nothing overdue. Suspicious." with reassuring subtitle. Subtle checkmark next to "0 tasks this week".
Digest nudge card. For skipped sign-in users. "Get a weekly summary in your inbox." with "Sign in to enable" CTA. Dismissible via X.
Gesture hint. "swipe right to complete" below first task card. 10px italic, colorMidAccessible. Disappears after first completion.
Standard Flutter implementation block · S-09
Mockup base:    390px width (iPhone 15)
SafeArea:       Wrap entire screen in SafeArea
Scroll:         SingleChildScrollView for content zone only (task list area)
Bottom zone:    Floating pill tab bar + FAB, positioned 14px from screen edge
Progress bar:   None (dashboard is post-onboarding)
Scaling:        All visual elements x (screenWidth/390).clamp(0.9, 1.15)
Special:        Tab bar is custom widget (NOT BottomNavigationBar). FAB is rounded square (15px radius, NOT circle). Swipe-to-complete uses Dismissible with 40% threshold. Greeting is time-aware (morning/afternoon/evening). Empty state hides category filter strip. Digest nudge card only shown to users who skipped sign-in.
