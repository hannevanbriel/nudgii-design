Design decisions: v3 flow changes
■
Entity cards replace tile grids
S-03 and S-04 now use full-width entity cards instead of 2- or 3-column tile grids. Cards show 28px icon circle, item name, meta line, and check circle (S-03) or expand chevron (S-04). Consistent with dashboard card language.
📅
Plan-style AHA screen
S-05 now shows month headers (October, November) with Plan-tab-style task rows: date column (day + number), category icon circle, task title, and item name. Percentage circles on month headers preview the Plan tab pattern.
💬
S-02 conversation moved to Ask tab
The conversation UI is no longer part of onboarding. Users go directly from S-01 Welcome to S-03 Browse. Conversation-based item adding lives in the Ask tab post-onboarding.
🔔
Push shows actual first date
"We'll remind you on Oct 28" makes the permission ask concrete. Abstract asks convert at 20-25%. Concrete asks convert at 60-70%.
🎨
Precision Warmth design language
Noto Serif headlines, Manrope body, warm surface (#FBF9F4), category-colored washes at 14% opacity, round icon circles, no borders on cards. Dusty rose, sage, sage-slate, lavender category palette.
✓
5 steps to dashboard
Welcome, Browse, Review, AHA, Push. Celebration + sign-in merged into S-05. One fewer screen than v2, two fewer than v1. User reaches dashboard faster without bait-and-switch moments.
