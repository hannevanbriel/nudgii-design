Done overlay — which trigger fires it?
Does the full-screen plum overlay fire from task detail "mark done", or only from the dashboard swipe-to-complete? If both, the overlay fires twice in one session.
Recommendation: dashboard swipe only. Task detail gets the quieter in-screen completion banner.
Affects: S-11, S-13, S-14
!
Ask tab — what does a first-time user see?
Three options: empty state with prompt suggestions · open voice input immediately · contextual suggestions based on upcoming tasks.
Affects: S-18, S-19 — product decision needed before these can be designed
!
Voice onboarding — does nudgii speak first?
Resolved 2026-03-21. nudgii never speaks. Voice = user input only, max 1 min/turn. nudgii always responds in text.
