Design language rules for all illustrations:
Only pill shapes (rounded rectangles with rx = half width) and circles/dots. No straight lines except ground bars. No literal object drawing (no house shape, no car shape, no leaf shape). Everything is abstract, composed from the same building blocks as the ii mark. Colors are exclusively from the nudgii token palette. Opacity varies between 0.1 and 0.7 to create depth without gradients. All SVGs are under 2KB. Hanne can add grain/noise texture in Figma as a final pass before PNG export for production.
